# 本项目用于解放大学习查询没做人员和已做人员，解放负责人的双手和时间！！！

## 1、准备你班级的数据,做成excel，复制路径到 bigstudy.py 的如下位置：
### 名单目录路径(格式看README.MD)
excel=''

Excel 格式（这俩列必须有，且‘是否完成大学习’这一列默认是‘否’）
|姓名|是否完成大学习 |
|张三|否            |
|李四|否            |
|王五|否            |
|... |否            |

## 2、安装需要的第三方库
pip install -r requirements.txt

## 3、输入自己的账号在 bigstudy.py 的如下位置
### 账号/密码/班级
account=''
password=''
your_class=''
